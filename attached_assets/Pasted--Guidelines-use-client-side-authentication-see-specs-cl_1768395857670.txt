# Guidelines

- use client side authentication, see @specs/client-side-auth-migration.md:
```
  useEffect(() => {
    if (authLoading) return;
    
    if (!user) {
      router.push('/login');
      return;
    }
    
    if (!hasAnyRole([UserRole.ADMIN, UserRole.CLUB_ADMIN])) {
      router.push('/');
    }
  }, [authLoading, user, hasAnyRole, router]);
```
- For all admin pages using PlayerForm:
-- Only UserRole.ADMIN and UserRole.CLUBADMIN may access the page and edit data.
-- Use apiClient from lib/apiClient.tsx for all HTTP requests.
-- Do not use axios directly.
-- Do not fetch users/me.
-- Use next/link’s Link component without deprecated <a> props.

- Keep the licence table and master data in sync with the backend:
-- The licence table must always reflect the current database state after any save, auto-optimize, revalidate or modal save.
-- “Master data” sections must also reflect the latest values after saving.
-- Every save operation must be clearly visible for the user with a success or error message.

## Create a new component TeamAssignmentSelect.tsx

- Data source:
-- Call GET /players/:id/possible-teams.
-- Use the data attribute of the returned document.
- Filtering:
-- Filter teams by clubId of the current user’s club (obtain from token payload).
-- Only show active teams of the own club.
- UI:
-- Dropdown listing available teams with:
--- Colored dot indicating status (valid/invalid)
--- Colored Badge indicating recommendedType
--- Example entry: "(colored dot) 1. Herren [SECONDARY]"
- Time-frame restriction:
-- Call GET /configs/PLAYERASSIGNMENTWINDOW.
-- Expected format in returned object:
--- Within the PLAYERASSIGNMENTWINDOW config:
---- key ENABLED, value true/false, sortOrder: 1
---- key STARTMONTH, value number (1–12), sortOrder: 2
---- key STARTDAY, value number (1–31), sortOrder: 3
---- key ENDMONTH, value number (1–12), sortOrder: 4
---- key ENDDAY, value number (1–31), sortOrder: 5
-- Logic:
--- If ENABLED is true and the current date is within the given window (inclusive) from STARTDAY.STARTMONTH to ENDDAY.ENDMONTH, then:
---- Enable the TeamAssignmentSelect dropdown.
---- Otherwise:
----- Disable the dropdown and visually indicate that the assignment window is closed (e.g. greyed out, tooltip).
