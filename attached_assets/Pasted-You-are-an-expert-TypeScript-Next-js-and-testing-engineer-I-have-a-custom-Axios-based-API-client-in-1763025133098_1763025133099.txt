### Current situation

- When a 401 occurs:
  - If there is a `refresh_token`, it calls the refresh endpoint and retries with the new token.
  - If refresh fails OR there is no `refresh_token`, it:
    - clears tokens from `localStorage` and
    - redirects to `/login` using `window.location.href = '/login'` or similar,
    - except if the current path is already `/login`.
